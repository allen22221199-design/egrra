# 煌盛興業 EGRRA 官網 — 使用說明

## 檔案
| 檔案 | 用途 |
|------|------|
| `index.html`   | 官網首頁（長捲，已內建右下角智能客服） |
| `products.html`| 花色庫頁（依系列 / 色系篩選，點花色看詳情 + 相關花色） |
| `cases.html`   | 實績案例牆（依類別 / 地區篩選，點案例看詳情） |
| `maintenance.html` | 維護模式頁（施工期間顯示） |
| `admin.html`   | 後台管理介面（改資訊 / 產品 / 案例 / 上傳圖片） |
| `site-data.js` | 網站內容資料（所有頁面與後台共用的預設內容） |
| `textures.js`  | 共用材質產生器（石紋 SVG 示意圖） |
| `lib.css`      | 花色庫 / 案例牆共用樣式 |
| `chatbot.js`   | 右下角智能客服（規則型；可接後端升級真 AI） |
| `api/chat.js`  | 真 AI 客服後端（Vercel/GCF Serverless，呼叫 Claude） |
| `package.json` | 後端相依套件（部署時 Vercel 會自動安裝） |
| `WORDPRESS.md` | 套用到既有 WordPress 站的相容性與移植規劃 |
| `README.md`    | 本說明 |

> 頁面關係：首頁的「瀏覽完整花色庫」「查看完整實績案例牆」按鈕會連到
> `products.html` / `cases.html`。這三頁 + 客服都讀同一份 `site-data.js`，
> 後台改內容會一起反映。

## 怎麼用後台
1. 用瀏覽器打開 `admin.html`（雙擊即可）。
2. 左側三個分頁：
   - **網站資訊**：電話、傳真、FB、首頁簡介、關於段落、經營理念。
   - **產品花色**：新增 / 編輯 / 刪除花色，可上傳產品照（會自動壓縮）。
   - **實績案例**：新增 / 編輯 / 刪除案例，可上傳現場照片。
3. 改完按右上 **「儲存變更」**。
4. 按 **「預覽官網」** 看效果。

## ⚠️ 最重要：資料存在哪、怎麼讓「所有人」看到
後台的修改**先存在你這台電腦的瀏覽器（localStorage）**。
你自己看官網會即時更新，但**別台裝置 / 其他訪客還看不到**。

要讓「全站訪客」都看到新內容，三種做法擇一：

1. **交給維護者部署（目前這版的做法）**
   後台按 **「匯出備份」** 會下載 `egrra-data-backup.json`，把它交給維護者（或我），
   更新到網站後重新部署即可全站生效。

2. **升級成線上 CMS（推薦，若你要常常自己更新）**
   改用 git 型 CMS（如 Decap CMS）或 WordPress，登入後台改完**自動發布**，
   不需每次找人。這需要一次性設定（GitHub / 主機串接）。

3. **接自架後端 + 資料庫**（客製化最高，維護成本也最高）。

## 部署上線（靜態站，最簡單）
把整個 `egrra-official` 資料夾拖到 **Vercel**（或 Netlify），幾秒就有網址；
再把網域 `egrra.com` 指向它即可。三個檔案都是純靜態，不需要 Node / build。

## 備份與還原
- **匯出備份**：下載目前所有內容成 JSON，建議定期存檔。
- **匯入備份**：把先前的 JSON 讀回來（會覆蓋目前內容）。
- **還原預設**：清除瀏覽器暫存即回到 `site-data.js` 的預設內容。

## 右下角智能客服（chatbot.js）
- 屬「規則型」小幫手：懂網站所有內容（產品、計價、保養、防火、客製、案例、聯絡），
  並自動讀 `site-data.js` / 後台資料，你後台加花色它就會一起講。
- 免後端、免費、純靜態，放上去就能用。
- **升級真 AI（已備好後端）**：`api/chat.js` 是現成的後端，用 Claude 回答。啟用步驟：
  1. 用 **Vercel** 部署整個資料夾（`api/chat.js` 會自動變成 `/api/chat`）。
  2. Vercel 專案 → Settings → Environment Variables 新增 `ANTHROPIC_API_KEY`（你的 Claude 金鑰）。
  3. 把 `chatbot.js` 最上方的 `AI_ENDPOINT` 改成 `"/api/chat"`，重新部署。
  - 之後「規則答不出」的問題會自動轉給 AI；常見問題仍由規則秒答（省錢）。
  - 費用：AI 訊息依 token 向 Anthropic 計費。想更省更快，把 `api/chat.js` 的
    `model` 改成 `"claude-haiku-4-5"`。金鑰只放後端環境變數，**絕不可放前端**。

## 小提醒
- 圖片上傳會自動壓縮到最長邊約 1400px；但瀏覽器暫存有容量上限（約 5MB），
  若放很多大圖可能存不下 —— 正式大量圖片建議走 CMS 或後端方案。
- 花色/案例沒有上傳照片時，官網會用「系統生成的材質示意圖」墊著。
